# Assets Directory

## Contents

This directory contains placeholder assets for the Savings Overview app.

### App Icon
- `AppIcon.appiconset/`: Placeholder app icon (1024x1024)
  - **Note**: Replace with actual app icon before production release

### Colors
- `AccentColor.colorset/`: Default accent color for the app
  - Current: Teal/Coral blend

### Adding Custom Assets

To add custom assets:

1. **App Icon**:
   - Replace the placeholder in `AppIcon.appiconset/`
   - Ensure 1024x1024 PNG without alpha channel

2. **Custom Colors**:
   - Add `.colorset` directories for themed colors
   - Define in `Contents.json` with RGB values

3. **Images**:
   - Add `.imageset` directories for custom images
   - Include @1x, @2x, @3x variants

### Bank Icons (Placeholder)

For bank/institution icons, use SF Symbols:
- `banknote`: Generic savings
- `building.columns`: Bank
- `creditcard`: Credit cards
- `dollarsign.circle`: Money/cash

### Recommended Assets to Add

- **Launch Screen**: Custom launch screen image
- **Empty State**: Illustrations for empty states
- **Onboarding**: Graphics for first-time user experience
- **Success/Error**: Icons for feedback states

### Asset Naming Convention

Follow these naming conventions:
- `icon_name_description` for icons
- `img_category_description` for images
- Use lowercase with underscores
- Be descriptive but concise

## Notes

All assets in this directory are placeholders and should be replaced with production-quality assets before app release.
